```
GET /merkle/block/:blockIndex
```
Returns Merkle root for specific block.

### 2. Get Election Merkle Root
```
GET /merkle/election/:electionId
```
Generate Merkle tree for all votes in election.

### 3. Generate Merkle Proof
```
POST /merkle/proof
Body: { transactionHash, electionId }
```
Generate proof for specific vote.

### 4. Verify Merkle Proof
```
POST /merkle/verify
Body: { vote, proof, merkleRoot }
```
Verify Merkle proof independently.

### 5. Get Merkle Statistics
```
GET /merkle/stats
```
Statistics for all elections.

### 6. Batch Verify Proofs
```
POST /merkle/batch-verify
Body: { items, merkleRoot }
```
Verify multiple proofs at once.
