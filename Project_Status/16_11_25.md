### If a test fails:

**Step 1: Isolate the Failure**
```bash

bash test-phase-X-task-Y.sh
```

**Step 2: Check Logs**
```bash

docker-compose logs blockchain-node-1 | tail -100
```

**Step 3: Assess Severity**
- Critical (data loss risk): Stop testing, restore backup
- High (feature broken): Fix and retest
- Low (timing issue): Note and continue

**Step 4: Recovery Options**
