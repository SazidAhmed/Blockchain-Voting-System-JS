# Word Export Steps (University Submission)

Use `Project_Status/PROJECT_REPORT_SUBMISSION_READY.md` as the base manuscript.

## 1) Open and Paste
1. Open Microsoft Word.
2. Create a new blank document.
3. Paste the full content of `PROJECT_REPORT_SUBMISSION_READY.md`.
4. Remove the literal `\newpage` text and insert actual page breaks (`Ctrl+Enter`) at those positions.

## 2) Apply Page Setup
1. Layout -> Size -> `A4`.
2. Layout -> Margins -> Custom:
   - Left: `1.5 in`
   - Right: `1.0 in`
   - Top: `1.0 in`
   - Bottom: `1.0 in`

## 3) Apply Fonts and Spacing
1. Select all body paragraphs:
   - Font: `Times New Roman`
   - Size: `12`
   - Alignment: `Justified`
   - Line spacing: `1.5`
   - Spacing After: `6 pt` (or university default)

## 4) Apply Heading Styles
1. Chapter titles (`Chapter 1`, `Chapter 2`, etc.) -> `Heading 1`.
2. Major section headings (`1.1`, `2.1`, etc.) -> `Heading 2`.
3. Subsections (if any deeper level) -> `Heading 3`.
4. Update style formatting:
   - Heading 1: 16 pt, bold
   - Heading 2: 14 pt, bold
   - Heading 3: 12 pt, bold

## 5) Auto TOC, Figures, Tables
1. Put cursor under "Table of Contents" -> References -> Table of Contents -> Automatic.
2. Add captions to each figure/table in Word:
   - References -> Insert Caption.
3. Insert List of Figures and List of Tables from References menu.

## 6) Page Numbering (Important)
1. Split document sections:
   - Front matter section (title page to abbreviations)
   - Main body section (Chapter 1 onward)
2. Front matter page numbers: Roman (`i, ii, iii`).
3. Main body page numbers: Arabic (`1, 2, 3`) and restart at `1`.
4. Disable "Link to Previous" between sections before changing format.

## 7) Final Quality Pass
1. Ensure all placeholders are replaced:
   - `[Your Name]`, `[Your ID]`, supervisor, department, date.
2. Run spelling + grammar check.
3. Update TOC, LOF, LOT (`Ctrl+A`, then `F9`).
4. Export PDF:
   - File -> Save As -> PDF.

## 8) Output Files
- `PROJECT_REPORT_SUBMISSION_READY.docx`
- `PROJECT_REPORT_SUBMISSION_READY.pdf`
